# 🚀 FUNDAE Manager - Landing Page B2B

Landing page de alta fidelidad para FUNDAE Manager (by Evoluxia), construida con Next.js, Tailwind CSS y Framer Motion.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Iconos**: Lucide React
- **Tipografías**: Plus Jakarta Sans + JetBrains Mono

## 🎨 Sistema de Diseño

- **Paleta de colores**:
  - Azul Noche: `#0A1118`
  - Azul Cobalto: `#2563EB`
  - Gris Perla: `#F8FAFC`
  - Carbón: `#0F172A`

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build producción
npm run build
```

## 🌐 Despliegue en Vercel

1. Sube este proyecto a un repositorio GitHub
2. Ve a [vercel.com](https://vercel.com) e importa el repositorio
3. Vercel detectará automáticamente Next.js
4. ¡Listo! Tu web estará publicada en segundos

## 📱 Ver en local

El servidor de desarrollo está disponible en:
```
http://localhost:3000
```
